      • Solution:
        • (x+y)n = Sum from k=0 to n of xk * y(n-k) * C(n,k)
        • Plug in x=2, y= 1. The result follows.
    • ii. give a combinatorial proof
      • Solution:
        • 3^ = 3 * 3 * .. * 3 - it counts the numbre of 3-colourings {Red,Blue,Green], of an n-element set.
        • Choose k (in C(n,k) ways) elements and suppose they are NOT going to be colored RED. The number of 2-colorings of this set is 2k. If we sum up 2k * C(n,k), over all k=0..n, we count all possible 3-colorings of the entire set.

        • Cn is equal to the number of walks from (0,0) to (n,n) with steps (0,1), (1,0)
        • Claim, there is a bijective map between the sequences of form 1 <= a1 <= a2 <= … <= an with ai <= i and the walks from (1,1) to (n+1,n+1) using steps from (0,1) and (1,0). s.t. the path never crosses x=y line.
        • Let f be the following.
        • f: W -> S, where W is the set of mentioned walks, and S is the set of the sequences.

        • Nice.
        • We limit the value ai can take by making sure the graph does not walk higher than y=x
    • Solution 2.
      • The following is a bijection between the sequences S and the sequences of n pluses and n minuses.
      • Let ai-1 be the number of minuses preceding the ith plus, put ai - ai-1 minuses after the (i-1)th plus.
        • This is like building a string, using + and -, and making sure that before any - at position i, there are no more than i-1 +es before it. It's like traversing a graph (box) right and up, as in solution 1.
        • There are n pluses in in all such sequences because of our construction (we add "+"s until nth). Why n '-'s? Because we have (an-1) '-'s preceding nth '+' and we added (n+1) after nth plus (at the tail). an-1 + (n+1 - an) = n.
